Smooks Webpage/Site report generation.

Package Specification

Collection of classes for using SmooksXML in standalone mode for generating a page/site test report.

Report on what?

Potential page/site markup and style elements that may result in browser incompatibility issues i.e. a component of markup/style on the page works fine on browser A but may cause issues on browser B. Basically, the Smoooks Report Generator is a web accessibility test tool.
