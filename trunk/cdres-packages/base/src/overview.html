Smooks Base Content Delivery Resources.

Overview

This module contains basic Content Delivery Resources reusable across all sorts of content manipulation operations.

>>>Download .cdrar<<<

See the contents of this packages META-INF folder below. These are the packages for preconfigured configurations. The Content Delivery Resources contained within this package are also available to other packages as long as this cdrar is loaded.

Packages Pre-Configurations

These are generic configurations that are perceived to be common across all browser types:
  • base-params.cdrl
  • common.cdrl
  • content-type.cdrl
  • param-decoders.cdrl
  • serialize-basic.cdrl
