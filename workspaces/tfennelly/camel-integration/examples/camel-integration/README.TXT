About This Example:
===================
 	This is a very simple example that illustrates the use integration of Smooks with Camel

	The following features are demonstrated:

	See src/main/resources/smooks-config.xml for inline comments.

    See:
        1. The "Main" class in src/main/java/example/Main.java.
        2. The input message in src/main/resourcesinput-message.jsn.
        3. src/main/resources/smooks-config.xml.

How to Run?
===========
    Requirements:
        1. JDK 1.5
        2. Maven 2.x (http://maven.apache.org/download.html)

    Running:
        1. "mvn clean install"
        2. "mvn exec:java"
