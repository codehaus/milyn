About This Example:
===================
    This is a very basic example that illustrates how Smooks can be
    used to apply a Java based transform on a message fragment. 

    See:
        1. The code in the src/main/java folder tree (.java).
        2. smooks-config.xml.

How to Run?
===========
    Requirements:
        1. JDK 1.5
        2. Maven 2.x

    Running:
        1. "mvn clean install"
        2. "mvn exec:java"
