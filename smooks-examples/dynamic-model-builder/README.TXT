About This Example:
===================
    This example illustrates how to use the ModelBuilder class
    to consume an XML message that contains multiple schema (xsd)
    namespaces and/or multiple schema versions.

How to Run?
===========
    Requirements:
        1. JDK 1.5+
        2. Maven 2.x (http://maven.apache.org/download.html)

    Running:
        1. "mvn clean install"
        2. "mvn exec:java"
