These pages are some simple examples of how to use the Tinak JSP Tag Library.

A basic "Hello World" sample: helloworld.jsp.

A more advanced "About Tinak" sample: about-tinak.jsp.

These sample pages were tested with the following browsers:
1. Internet Explorer v6.0
2. Mozilla Firefox v1.0.2
3. Openwave SDK v6.2.2
4. Pocket PC 2003 Emulator (Emulator for Windows CE - v4.2.0.20)
